**Denis Escape** adalah sebuah game berbasis **Unreal Engine 5** yang mengharuskan pemain membantu karakter bernama **Denis** untuk menemukan kunci dan keluar dari sebuah labirin.

Dalam permainan ini, Denis harus menyusuri labirin sambil menghindari ancaman sebuah **robot pengejar**. Robot tersebut dilengkapi dengan senter, dan apabila Denis terkena cahaya senter tersebut, maka permainan akan **langsung berakhir (Game Over)**.

Tujuan utama pemain adalah mencari kunci, menghindari robot, dan menemukan jalan keluar dari labirin dengan selamat.
